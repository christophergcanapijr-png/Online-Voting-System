# 🗳️ Online Voting System

A web-based online voting system built as a capstone project for my IT degree.

## 📌 About the Project
This system allows students to vote for their school organization candidates online. It features a secure admin panel and a clean voting interface.

## ✨ Features
- 🔐 Secure student login
- 🗳️ Online voting for candidates
- 📊 Real-time vote counting & analytics
- 👨‍💼 Admin panel for managing voters & candidates
- 📁 Import voters via CSV
- 📄 Export reports to Excel
- 🏆 Canvassing report generation

## 🛠️ Built With
- PHP
- MySQL
- HTML & CSS
- JavaScript
- Bootstrap
